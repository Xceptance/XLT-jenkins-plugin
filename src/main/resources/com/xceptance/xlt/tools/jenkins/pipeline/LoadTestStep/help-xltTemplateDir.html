Specify the path to the XLT installation directory. This path can be relative to the Jenkins root directory or absolute. Environment variables can be accessed by using the ${<variableName>} syntax. For example "${HOME}/xlt".

This installation acts as a template and will be copied to a temporary location for each load test. To accelerate this step, remove any sub directory not necessary for load testing, such as doc, samples, or tools, from the installation folder.
