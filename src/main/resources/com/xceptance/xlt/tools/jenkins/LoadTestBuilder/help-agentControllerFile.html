Edit the file which is used for editing the Agent Controller URLs. The file have to be within the testsuite in the config directory.

In the file use the following notation:

<protocol>://<hostname>:<port>

For seperating multiple URLs you can use: '\n'|'\s'|','|';'

For example an EC2 instance of Amazon Web Services could be https://ec2-54-196-32-164.compute-1.amazonaws.com:8500. If nothing is listed XLT switches to -embedded mode.
