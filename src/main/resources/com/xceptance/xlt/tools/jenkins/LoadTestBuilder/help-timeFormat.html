This option allows you to customize the date/time presentation in the charts as a pattern. If empty, the default will be used. The pattern "' '" (Without the double quotes) can be used to hide the date/time, for example if you just want to see the build number (see option "Show Build Number").

Within the pattern, unquoted letters from 'A' to 'Z' and from 'a' to 'z' are interpreted as pattern letters. Text can be quoted using single quotes (') to avoid interpretation. "''" represents a single quote. All other characters are not interpreted; they're simply copied to the output string.

For a full explanation of the pattern syntax see here.

Examples

Pattern Result
yyyy.MM.dd 'at' HH:mm:ss z 2014.07.25 at 15:17:30 EDT
yy/MM/dd-HH:mm 14/07/25-15:17
h:mm a 3:17 PM
HH:mm 15:17
