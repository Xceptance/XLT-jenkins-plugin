This option allows you to customize the date time presentation in the charts. If empty, the default will be used. The pattern "' '" (Without the double quotes) can be used to hide the date time, for example if you just want to see the build number (See option "Show Buildnumber").

Within the pattern, unquoted letters from 'A' to 'Z' and from 'a' to 'z' are interpreted as pattern letters. Text can be quoted using single quotes (') to avoid interpretation. "''" represents a single quote. All other characters are not interpreted. they're simply copied into the output.
