Specify the name of the file that contains the URLs of the agent controllers to use for the load test. The file will be searched in the config directory of your test suite project. In the file, use the following notation:

https://<hostname1>:<port1> ; https://<hostname2>:<port2> ; ...

To separate multiple URLs you can use: '\n'|' '|','|';'

If the file is empty, the load test cannot be started.
